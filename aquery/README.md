# Local aquery Docker images

A licensed version of kdb+ is required. Put the base64-encoded license in the repository-root `.env` file:

```env
KDB_LICENSE_B64=your_base64_encoded_kdb_license
```

Choose the setup for your host:

- [x86-64 Linux](docker-x86-linux/README.md)
- [macOS with Docker Desktop](docker-mac/README.md)
